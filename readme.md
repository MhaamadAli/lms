# Learning Management System (LMS)

A desktop application built with JavaFX, designed to facilitate online learning through a streamlined, user-friendly interface.

## Features

- User Management
- Course Management
- Assignment Submission
- Quiz Functionality
- Progress Tracking

## Technology Stack

- Frontend: JavaFX
- Backend: Java
- Database: MySQL
